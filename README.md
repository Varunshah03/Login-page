Project Description: Responsive Sign-Up Page
This project is a responsive sign-up page designed with HTML and CSS. It features a clean and modern layout, making it suitable for user registration forms in web applications. The page is fully responsive and adapts to different screen sizes, ensuring a seamless user experience across devices.

Key Features:

Form Section: Includes input fields for full name, email, and password, along with a submit button.
Social Login Buttons: Provides options for signing up using Google or Facebook.
Illustration Section: Displays an image alongside the form to enhance visual appeal.
Responsive Design: The layout adjusts to fit various screen sizes, ensuring usability on desktops, tablets, and mobile devices.
Validation: Includes basic email validation using JavaScript to ensure proper input.

This project is ideal for learning or demonstrating front-end development skills, including responsive design, form handling, and basic JavaScript validation.
